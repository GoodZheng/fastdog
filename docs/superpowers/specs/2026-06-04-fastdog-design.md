# FastDog - 文本搜索工具设计文档

## 概述

FastDog 是一个基于 C# WPF 的文本搜索工具，参考 grepWin 的核心搜索功能，使用 ripgrep 作为搜索引擎后端，面向团队内部使用。

## 功能范围

### 保留的功能

- 纯文本搜索
- 正则表达式搜索
- 大小写敏感/不敏感
- 全词匹配
- 文件名通配符过滤（如 `*.cs;*.txt`）
- 排除目录（如 `bin;obj;.git`）
- 按修改日期范围过滤
- 搜索结果：文件列表 + 匹配行预览
- 双击打开文件、复制路径、跳转行号

### 删除的功能（相对 grepWin）

- 替换功能
- 书签管理
- Shell 右键菜单集成（ExplorerCommandVerb）
- 暗色模式
- 多目录搜索
- 多语言支持
- 便携模式
- 编码检测/转换
- 换行符转换
- 自动更新
- 关于对话框中的捐赠链接
- 文件大小过滤
- 二进制文件搜索
- 反向搜索（找不匹配的文件）

### 三层职责

- **UI 层**：WPF MainWindow，MVVM 模式（CommunityToolkit.Mvvm）
- **业务层**：SearchService，将用户输入翻译为 rg 命令参数，处理日期过滤等 rg 不支持的功能
- **进程层**：RipgrepBridge，管理 rg.exe 进程生命周期和 JSON 输出解析

## UI 布局

### 搜索条件区（上方，紧凑两行布局）

```
┌──────────────────────────────────────────────────────────────┐
│ 搜索路径: [E:\code\project                    ] [浏览...]    │
│ [搜索内容.......................] ○正则 ○文本 ☑区分大小写 ☑全词 │
├──────────────────────────────────────────────────────────────┤
│ 文件过滤: [*.cs;*.txt     ] 排除目录: [bin;obj;.git  ]       │
│ 修改日期: [____/__/__]~[____/__/__] ☐启用        [搜索][取消] │
└──────────────────────────────────────────────────────────────┘
```

- 搜索路径：手动输入 + 浏览按钮 + 拖入文件夹
- 搜索类型：正则/纯文本二选一（RadioButton）
- 大小写敏感：默认不敏感，勾选后启用
- 全词匹配：对应 rg 的 `-w`
- 文件过滤：分号分隔多个通配符，对应 rg 的 `-g` 参数
- 排除目录：分号分隔，对应 rg 的 `-g !{dir}` 参数，内置默认排除 `.git`
- 日期范围：勾选启用后生效，由业务层二次过滤
- 取消按钮：中断搜索（Kill rg 进程），已显示的结果保留

### 搜索结果区（下方）

**文件列表（DataGrid）**：

| 文件名 | 大小 | 匹配数 | 路径 | 修改时间 |
|--------|------|--------|------|----------|

- 支持列排序、列宽拖动
- 双击 → 系统关联程序打开文件
- 右键菜单：复制路径、打开文件、复制文件名

**行内容预览（下方）**：

- 点击文件后显示该文件所有匹配行
- 格式：`行号: 匹配行内容`，关键词高亮
- 点击某行 → 编辑器打开并跳转到行号

**状态栏（底部）**：

- 搜索状态（空闲/搜索中）、匹配文件数、总匹配数、耗时

## ripgrep 集成

### 命令参数映射

| 用户操作 | rg 参数 |
|---------|---------|
| 正则搜索 | 默认行为 |
| 纯文本搜索 | `-F` |
| 大小写敏感 | 默认不敏感，勾选后不加 `-i` |
| 全词匹配 | `-w` |
| 文件过滤 | `-g *.cs -g *.txt` |
| 排除目录 | `-g !{bin,obj}` |
| JSON 输出 | `--json` |

### 输出解析

- 使用 `--json` 模式，每行一个 JSON 对象
- 异步读取 `Process.StandardOutput`，逐行解析，实时推送结果到 UI
- JSON 中包含文件路径、行号、匹配文本、匹配偏移量

### 日期过滤

rg 不支持日期过滤，实现方式：
1. rg 搜索得到全部匹配文件
2. 遍历结果，用 `FileInfo.LastWriteTime` 过滤
3. 不符合日期范围的结果不显示

### 进程管理

- 启动：`Process.Start` 启动 rg.exe
- 取消：`Process.Kill` 终止
- rg.exe 位置：应用目录下 `tools/rg.exe`

## 文件打开

- 系统关联打开：`Process.Start(new ProcessStartInfo(filePath) { UseShellExecute = true })`
- 跳转行号：尝试 `code --goto file:line`（VS Code），否则仅打开文件
